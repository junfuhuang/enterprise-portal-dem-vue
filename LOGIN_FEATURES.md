# 登录模块功能说明

## 📋 概述

为企业门户网站添加了完整的登录模块，包括用户认证、状态管理和个人中心功能。

## ✨ 新增功能

### 1. **登录弹窗组件** (`components/Login.vue`)
- ✅ 精美的模态弹窗设计
- ✅ 邮箱和密码表单验证
- ✅ 显示/隐藏密码切换
- ✅ 加载状态显示
- ✅ "记住我"功能
- ✅ 忘记密码链接
- ✅ 快速演示账户登录
- ✅ 注册入口

### 2. **用户状态管理** (`App.vue`)
- ✅ Vue 3 Composition API
- ✅ localStorage 持久化存储
- ✅ 页面刷新保持登录状态
- ✅ 登录/登出事件处理

### 3. **导航栏增强** (`components/Header.vue`)
- ✅ 未登录状态：显示"登录"和"注册"按钮
- ✅ 已登录状态：
  - 显示用户头像（首字母）
  - 显示用户名
  - 下拉菜单（个人中心、设置、退出登录）
- ✅ 响应式设计（桌面端和移动端）

## 🎨 界面特点

- **现代化设计**：圆角、阴影、渐变色
- **流畅动画**：淡入淡出、缩放效果
- **表单验证**：实时错误提示
- **无障碍**：符合 ARIA 标准
- **响应式**：完美适配各种设备

## 🔐 使用方式

### 快速体验
点击"使用演示账户登录"按钮即可快速登录

### 自定义登录
1. 输入邮箱地址（格式：xxx@xxx.xxx）
2. 输入密码（至少6个字符）
3. 可选：勾选"记住我"
4. 点击"登录"按钮

### 演示账户
- **邮箱**：demo@example.com
- **密码**：demo123

## 📦 技术栈

- **Vue 3** - UI 框架
- **TypeScript** - 类型安全
- **Lucide Vue** - 图标库
- **Tailwind CSS** - 样式框架
- **localStorage** - 本地存储
